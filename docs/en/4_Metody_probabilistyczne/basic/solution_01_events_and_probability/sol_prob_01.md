# Solution
# Task 1 — Coin Tossing

Consider an experiment consisting of tossing a fair coin where the order of outcomes matters.

## Sample space for one coin toss
Ω₁ = {H, T}

Number of outcomes:
|Ω₁| = 2

## Sample space for two coin tosses
Ω₂ = {(H,H), (H,T), (T,H), (T,T)}

Number of outcomes:
|Ω₂| = 4

## Sample space for three coin tosses
Ω₃ = {
(H,H,H),
(H,H,T),
(H,T,H),
(H,T,T),
(T,H,H),
(T,H,T),
(T,T,H),
(T,T,T)
}

Number of outcomes:
|Ω₃| = 8

## Elementary outcome

An elementary outcome represents one specific ordered sequence of coin toss results.

Example:
(H,T) means the first toss is Heads and the second toss is Tails.
